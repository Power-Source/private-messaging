== Description ==
Private Benutzer-zu-Benutzer-Kommunikation zur Abgabe von Angeboten, zum Teilen von Projektspezifikationen und zur internen Kommunikation. Kommt mit Front-End-Integration, geschuetzten Kontaktinformationen und geschuetzter Dateifreigabe.

== Highlights ==
- Front-End-Messaging: Zugriff auf den Posteingang ueber das Mail-Symbol in der Admin-Toolbar, Nachrichten von jeder Seite senden und Empfaenger automatisch vervollstaendigen.
- Private Dateifreigabe: Fotos, Screenshots und Dateien per privater Nachricht teilen; Freigabe nach Benutzerrolle steuerbar.
- Einfache Einrichtung: Plugin installieren/aktivieren, Posteingangsseite per Klick unter Messaging -> Settings erstellen.
- Archiv & Einstellungen: Nachrichten archivieren oder zurueckholen; Lesebestaetigungen aktivieren/deaktivieren und weitere Optionen direkt im Posteingang.
- Add-ons: bbPress-Integration, Block List, Capability-Steuerung, Benachrichtigungen (Beta) und WYSIWYG-Editor.

== Installation ==
1. Plugin hochladen und aktivieren.
2. Im Dashboard zu Messaging -> Settings wechseln.
3. Unter "Inbox Page" eine vorhandene Seite waehlen oder "Create Page" nutzen und Aenderungen speichern.
4. Ueber das Mail-Symbol in der Admin-Bar "View Inbox" aufrufen.

== Bedienung ==
- Neue Nachricht: Im Posteingang auf "Compose" klicken oder in der Admin-Bar "Send New Message" waehlen.
- Archiv: Nachrichten ueber das Archiv-Symbol ablegen; spaeter aus dem Tab "Archives" zurueck in den Posteingang holen oder endgueltig loeschen.
- Einstellungen: Im Posteingang Lesebestaetigungen ein-/ausschalten und Blocklisten pflegen (falls Add-on aktiv).

== Add-ons ==
- BBPress integration: "Message Me"-Button in Threads fuer Direktnachrichten.
- Block List: Nachrichten von bestimmten Nutzern blockieren.
- Capability: Versand von Nachrichten auf definierte Rollen beschraenken (Messaging -> Settings -> Capability Settings).
- Notification (Beta): Pop-up-Benachrichtigungen fuer neue Nachrichten.
- WYSIWYG: Ersetzt den Standardeditor durch einen erweiterten Editor fuer formatierte Nachrichten.

== Einstellungen ==
- General: Lesebestaetigungen steuern und (sofern erlaubt) vom Nutzer deaktivierbar machen.
- Email: Betreffzeilen und Inhalte fuer E-Mail-Benachrichtigungen anpassen; Anzahl E-Mails pro Seite festlegen.
- Shortcodes: In den Einstellungen einsehbar, inklusive Parameter.
- Attachments: Festlegen, welche Rollen Dateianhaenge senden duerfen.
